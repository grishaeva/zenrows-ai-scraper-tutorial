# AI-Assisted Web Scraper Tutorial (ZenRows Integration)

This repository provides a hands-on tutorial for building a robust web scraper. It demonstrates how to combine AI-driven parsing with the ZenRows API to bypass modern anti-bot systems effectively.

## 📖 Methodology
This tutorial is designed following the [Diátaxis framework](https://diataxis.fr/), focusing on a **learning-oriented** approach. It provides actionable steps, clear prerequisites, and a tangible outcome.

## 🚀 Getting Started

### Prerequisites
- Python 3.x
- [ZenRows API Key](https://app.zenrows.com/)
- `pip install requests beautifulsoup4`

### Usage
1. Clone this repository: `git clone https://github.com/grishaeva/zenrows-ai-scraper-tutorial.git`
2. Update the `scraper.py` file with your `API_KEY`.
3. Run the scraper:
   ```bash
   python scraper.py

## 🛠 Features
- **AI-Driven Parsing:** Demonstrates how to automate data extraction logic using LLM-generated selectors (e.g., ChatGPT or Claude), making the scraper adaptable to site changes.
- **Anti-Bot Resilience:** Integrates the ZenRows API to handle complex challenges like proxy rotation, User-Agent spoofing, and JavaScript rendering automatically.
- **Diátaxis-Compliant:** Structured specifically as a "Tutorial" (learning-oriented) to ensure a frictionless developer experience.
